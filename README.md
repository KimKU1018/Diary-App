# Diary-App
일기장 앱 입니다.

# 기능 상세
- 일기장 탭을 누르면 일기 리스트 표시
- 즐겨찾기 탭을 누르면 즐겨찾기한 일기 리스트를 표시
- 일기를 등록, 수정, 삭제, 즐겨찾기 할 수 있음

# 활용 기술
- UITabBarController
- UICollectionView
- NotificationCenter
